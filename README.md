# LeetCodeTestbed
A C++ code template for debugging leetcode solutions.

## Getting start

1. Copy your solution code to `solution.h`
1. Write test code in `main.cpp`

## Input Helper

```cpp
// For vector<T>
// [1, 2, 3]
vector<T> vec = {1, 2, 3};


// For LinkedList
// 1->2->3
LinkedList list = {1, 2, 3};


// For TreeNode
//    1
//  /  \
// 2    3
//  \
//   4
TreeNode tree("[1, 2, 3, null, 4]");

```
